# vue-calendar-simple

## 安装
```
npm install vue-calendar-simple --save
```

## 全局引入
```
import Vue from 'vue'
import VueCalendarSimple from 'vue-calendar-simple'
Vue.use(VueCalendarSimple)
```
## 组件引入
```
import Vue from 'vue'
import { VueCalendarSimple } from 'vue-calendar-simple'
export default {
  components: {
    VueCalendarSimple
  }
}
```
